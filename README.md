# HH Clicker - Автоматическое повышение активности на HeadHunter

Приложение для автоматического просмотра вакансий на сайте hh.ru с целью повышения рейтинга активности до 100%.

## Функциональность

- Авторизация на сайте hh.ru
- Поиск вакансий по ключевому слову
- Автоматическое открытие карточек вакансий
- Отслеживание уровня активности
- Продолжение просмотра вакансий до достижения 100% активности

## Установка

1. Клонируйте репозиторий:
```bash
git clone <your-repo-url>
cd hh-clicker
```

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env` на основе `.env.example` и укажите свои учетные данные:
```bash
cp .env.example .env
# Затем отредактируйте .env и укажите свои данные
```

4. Запустите приложение:
```bash
npm run dev
```

## Использование

1. Откройте приложение в браузере по адресу http://localhost:3000
2. Введите query для поиска вакансий
3. Нажмите "Начать просмотр" для запуска процесса автоматического просмотра вакансий
4. Наблюдайте за прогрессом и статусом активности

## Конфигурация

Параметры работы приложения можно настроить в файле `src/lib/hh-scraper.ts`:

- `delayBetweenViews`: задержка между открытием вакансий (в миллисекундах)
- `maxRetries`: максимальное количество попыток при ошибках

## Безопасность

Важно: файл `.env` содержит чувствительные данные и не должен коммититься в репозиторий.

## Технические детали

Приложение использует:
- Next.js для веб-интерфейса
- Playwright для автоматизации браузера
- TypeScript для строгой типизации
