# HH Clicker - Инструкция по запуску и использованию

## Обзор проекта

HH Clicker - это Next.js приложение с использованием Playwright для автоматического просмотра вакансий на сайте hh.ru. Приложение автоматически входит в аккаунт, ищет вакансии по ключевому слову "java" и открывает карточки вакансий до достижения 100% рейтинга активности.

## Структура проекта

```
hh-clicker/
├── src/
│   ├── pages/
│   │   └── index.tsx          # Главная страница управления

│   ├── types/
│   │   └── hh-types.ts        # Типы данных для HH
│   └── utils/
├── playwright/
│   └── tests/
│       └── hh-flow.spec.ts    # Тесты сценариев работы
├── public/
├── package.json
├── next.config.js
├── tsconfig.json
├── playwright.config.ts
├── README.md
└── INSTRUCTIONS.md
```

## Требования

- Node.js версии 16 или выше
- npm или yarn

## Установка и запуск

### 1. Клонирование и установка зависимостей

```bash
git clone <your-repo-url>
cd hh-clicker
npm install
```

### 2. Создание .env файла

Создайте файл `.env` на основе `.env.example` и укажите свои учетные данные:

```bash
cp .env.example .env
# Затем отредактируйте .env и укажите свои данные
```

Важно: файл `.env` содержит чувствительные данные и не должен коммититься в репозиторий.

### 3. Запуск приложения

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

### 3. Запуск тестов Playwright

```bash
npm run test
```

## Использование приложения

### Авторизация

1. Откройте приложение в браузере
2. Учетные данные для входа на hh.ru будут взяты из переменных окружения
3. Нажмите "Войти"

### Запуск процесса сканирования

1. После успешной авторизации нажмите кнопку "Начать просмотр"
2. Приложение автоматически:
   - Войдет на сайт hh.ru
   - Найдет вакансии по запросу "java"
   - Будет открывать карточки вакансий с задержками
   - Отслеживать уровень активности
   - Продолжать до достижения 100% активности

### Остановка процесса

Вы можете остановить процесс в любое время, нажав кнопку "Остановить".

## Настройка параметров

Вы можете настроить параметры работы в файле `src/lib/hh-scraper.ts`:

- `delayBetweenViews`: задержка между открытием вакансий (в миллисекундах)
- `maxRetries`: максимальное количество попыток при ошибках
- `viewport`: размер окна браузера

## Безопасность

- Приложение не передает учетные данные на сторонние серверы

## Технические особенности

### Playwright

- Используется для автоматизации браузера
- Имитирует действия реального пользователя
- Обходит базовую защиту от ботов

### Отслеживание активности

- Приложение анализирует DOM элементы для определения уровня активности
- Использует различные стратегии поиска информации об активности
- Обновляет прогресс в реальном времени

### Обработка ошибок

- Повторные попытки при сетевых ошибках
- Корректная обработка закрытия вкладок
- Логирование ошибок для диагностики

## Ограничения и предостережения

1. Используйте приложение ответственно и в соответствии с условиями использования HeadHunter
2. Частые автоматические действия могут вызвать подозрения системы безопасности HH
3. Рекомендуется использовать значительные задержки между действиями для имитации реального поведения пользователя
4. Приложение работает с сайтом hh.ru и может потребовать обновлений при изменении структуры сайта

## Возможные проблемы и решения

- Если приложение не может определить уровень активности, проверьте, не изменился ли формат отображения на сайте HH
- При блокировке IP проверьте настройки задержек и частоты запросов
- При ошибках авторизации убедитесь, что учетные данные введены правильно

## Заключение

HH Clicker предоставляет удобный способ автоматического просмотра вакансий для повышения рейтинга активности на HeadHunter. Приложение разработано с учетом безопасности и надежности, но его использование остается на усмотрение пользователя.
